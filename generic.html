GitHub Projects

A Collection of GitHub Projects I have worked on
click on the headers to be directed to the project

RouterConfigpkt.py

Project written in python to be used with Cisco Packet Tracer for quick configuration of Routers and Switches. This project was created to simplify and automate the process of networking, as it reduced time for configuring multiple Routers and Switches from hours to minutes.

modRockyou.py

Simplified python script to extract an equal amount, less than an amount, or greater than an amount of characters from the rockyou.txt file. This is helpful in CTFs where the password policy is known, and shortens the amount of time needed for bruteforcing and hash cracking, by filtering known failed conditions.

SecTools

Identity Access Management script written in python, utilizing libraries such as: requests, selenium, and pandas. I created this script to automate a repetitive process, that in which saved approximately 20 work hours of repetitive work.

Gam_Gam

Python script to be used with GAMADV-XTD3 for automation of certain properties that exist within the Google Workspace platform. The Primary purpose was for configuration of security group management.

Animation config helper - Flipper Zero

Python script to be used with the Flipper Zero for helping with creating manifest.txt files for custom animations. This simplifies adding multiple animation folders and setting their configuration instructions.
